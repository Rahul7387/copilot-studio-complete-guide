# Pre-Deployment Checklist

## Instructions
- [ ] Role and scope clearly defined
- [ ] Do-not-do rules listed
- [ ] Escalation behavior defined

## Knowledge
- [ ] SharePoint connected and synced
- [ ] File collection uploaded
- [ ] Tested with 10+ real questions
- [ ] Unanswered questions reviewed

## Topics
- [ ] Password Reset tested end-to-end
- [ ] VPN Setup tested for all device types
- [ ] Raise Ticket connected to Power Automate flow
- [ ] Escalate configured with live agent handoff
- [ ] End of Conversation topic active (for CSAT)

## Tools
- [ ] Power Automate flow tested independently
- [ ] Tool connected and tested in agent
- [ ] Error handling tested

## Evaluation
- [ ] Test set imported and run
- [ ] Pass rate 90%+
- [ ] Failed cases fixed

## Security
- [ ] Azure AD authentication configured
- [ ] No sensitive data exposed in responses

## Channel
- [ ] Published to Microsoft Teams
- [ ] Tested on desktop and mobile
- [ ] Pilot group tested

## Analytics
- [ ] End of Conversation active
- [ ] Thumbs up/down enabled
- [ ] First review scheduled (1 week post-launch)
